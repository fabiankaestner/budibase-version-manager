# Usage
<!-- usage -->
```sh-session
$ npm install -g budibase-version-manager
$ bbvm COMMAND
running command...
$ bbvm (-v|--version|version)
budibase-version-manager/0.0.1 win32-x64 node-v14.17.3
$ bbvm --help [COMMAND]
USAGE
  $ bbvm COMMAND
...
```

## `bbvm help [COMMAND]`

display help for bbvm

```
USAGE
  $ bbvm help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
