# Usage
<!-- usage -->
```sh-session
$ npm install -g @budibase-version-manager/cli
$ bbvm COMMAND
running command...
$ bbvm (-v|--version|version)
@budibase-version-manager/cli/0.0.1 win32-x64 node-v14.17.3
$ bbvm --help [COMMAND]
USAGE
  $ bbvm COMMAND
...
```
